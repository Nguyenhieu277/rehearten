# ReHearten Static Images

Thư mục này chứa các hình ảnh và logo cho ReHearten:

## Cần có các file:
- `logo_rehearten.png` - Logo chính của ReHearten

- `nen.jpg` - Hình nền cho background
- `illus.mp4` - Video background (nếu có)
